## 포켓몬 도감

### 기능 

* 포켓몬 List 조회
* 각 능력치에 따른 filter 가능
* 찾고자 하는 포켓몬 이름, 도감번호로 검색 기능
* 자신만의 포켓몬 도감 작성 가능
___
### Tech
>* React(React Hook) + Typescript 기반
>* React-Router 사용
>* custom hook - useInput(input 기능), useWindowSize(window size 반응), useBookList(상태관리 booklist 재사용 위해 제작)
>* 상태관리 Redux(toolkit) 활용
>* css 전처리기 사용(scss)
